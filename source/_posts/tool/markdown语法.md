## 常用

### 文字

- 使用 `*` 或 `_` 包围文字表示斜体，如 `*斜体文字*` 或 `_斜体文字_`。
- 使用 `**` 或 `__` 包围文字表示加粗，如 `**粗体文字**` 或 `__粗体文字__`。
- 使用 `~~` 包围文字表示删除线，如 `~~删除线文字~~`。

### 超链接

```txt
[百度](http://www.baidu.com "百度搜索")
```

[百度](http://www.baidu.com "百度搜索")

### 字体颜色

#### 方式一

```python
<font color="red">这是红色的文字</font>
```

<font color="red">这是红色的文字</font>

#### 方式二

```python
`改变颜色` 
英文输入法下的左上角的~键
```

`改变颜色` 

#### 方式三

```python
$\color{red}{这是红色的文字}$
```

$\color{red}{这是红色的文字}$
