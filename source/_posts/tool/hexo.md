### 安装

```sh
npm install -g hexo-cli
```

### 下载包

```sh
npm install
```

### 清空

```sh
hexo clean
```

### 生成内容

```sh
hexo g
```

### 启动项目

```sh
hexo s
```

### 打包

public下的文件为打好的包
